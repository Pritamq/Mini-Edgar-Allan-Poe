# Mini-Edgar-Allan-Poe
Implemented a mini LLaMA-style autoregressive Transformer from scratch in PyTorch, including Byte-Level BPE tokenization, RMSNorm, RoPE, Grouped Query Attention, SwiGLU feed-forward layers, causal masking, AdamW training, validation monitoring, and top-p text generation on an Edgar Allan Poe corpus.
